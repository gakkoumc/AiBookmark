# AiBookmark

Google Chromeの「ブックマークバー」を扱いやすいYAMLフォーマットで一括編集するためのCLIツールです。
ブックマークが増えすぎて整理が追いつかなくなった際に、テキストエディタでまとめて階層移動や削除、追加を行うことができます。

## 準備

Node.js環境が必要です。
依存パッケージをインストールします。
```bash
npm install
```

## 使い方

### 1. エクスポート (Chrome -> YAML)
以下のコマンドを実行すると、現在のChromeのブックマークバーの内容が `bookmarks.yaml` に出力されます。
```bash
node index.js export
```

### 2. 編集
出力された `bookmarks.yaml` をお好きなテキストエディタで編集します。
- `type: url` や `type: folder` などを維持してください。
- インデントを使って階層を表現します。

### 3. インポート (YAML -> Chrome)

> **【重要】**
> インポート処理を行う前には、**必ずGoogle Chromeを完全に終了させておいてください**。バックグラウンドプロセスが残っていると、Chromeが変更を上書きして元に戻してしまう可能性があります。

編集後、以下のコマンドでChromeのブックマークに反映させます。
```bash
node index.js import
```

※実行時、同じディレクトリに元のブックマークファイルのバックアップ（`Bookmarks.bak`）が自動作成されます。万が一破損した場合は、手動で復元してください。
