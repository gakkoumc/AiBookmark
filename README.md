# AiBookmark

AiBookmark は、Chrome のブックマークを YAML 形式でエクスポート／インポートできる拡張機能です。
テキストエディタで階層をまとめて編集し、ブックマークバーへ反映できます。

## 主な機能
- ブックマークバーの構造を YAML としてエクスポート
- YAML を編集してフォルダ／URL の階層を変更
- 可能な限り既存 ID を維持するスマートマージでインポート
- YAML に存在しない孤立ノードを整理

## プロジェクト状態
このリポジトリは OSS 公開向けに、ライセンス・貢献ガイド・セキュリティ方針を整備しています。

## インストール（開発者モード）
1. このリポジトリを clone します。
2. Chrome で `chrome://extensions/` を開きます。
3. **デベロッパー モード** を有効化します。
4. **パッケージ化されていない拡張機能を読み込む** をクリックします。
5. このリポジトリのディレクトリを選択します。

> 注: 読み込み時には `manifest.json` がディレクトリ内に存在することを確認してください。

## 使い方
1. AiBookmark のポップアップを開きます。
2. **Export YAML** を押して現在のブックマークを出力します。
3. YAML を編集します。
4. **Import YAML** を押して変更を反映します。

## プライバシーとデータ取り扱い
- AiBookmark は Chrome Bookmarks API を使ってローカルで動作します。
- ブックマークデータを外部サーバーへ送信しません。
