# コントリビューティングガイド

AiBookmark への貢献に興味を持っていただきありがとうございます。

## 開発セットアップ
1. リポジトリを fork / clone します。
2. `chrome://extensions` の開発者モードで拡張機能を読み込みます。
3. 機能ごとにブランチを作成して開発してください。

## プルリクエスト
- 変更は可能な限り小さく、目的を明確にしてください。
- 振る舞い変更がある場合は、変更内容を具体的に説明してください。
- UI の変更がある場合は、可能であればスクリーンショットを添付してください。
- 振る舞いを変える場合は、関連ドキュメントも更新してください。

## コード方針
- 可読性を優先し、不要な複雑化を避けてください。
- YAML 形式の後方互換性を可能な限り維持してください。

## バグ報告に含めてほしい情報
- 再現手順
- 期待される結果
- 実際の結果
- （必要に応じて）問題を再現する YAML サンプル
- Chrome のバージョン
