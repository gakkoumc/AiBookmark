# Contributing

Thanks for your interest in contributing to AiBookmark.

## Development setup
1. Fork and clone the repository.
2. Load the extension in Chrome via `chrome://extensions` (Developer mode).
3. Make your changes in a feature branch.

## Pull requests
- Keep PRs focused and small.
- Include a clear description of behavior changes.
- If UI behavior changes, include screenshots where possible.
- Update docs when behavior changes.

## Code style
- Keep logic readable and avoid unnecessary complexity.
- Preserve backward compatibility of YAML shape where possible.

## Reporting bugs
Please include:
- Reproduction steps
- Expected behavior
- Actual behavior
- Sample YAML (if relevant)
- Chrome version
